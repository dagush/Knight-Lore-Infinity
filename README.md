# Knight Lore Infinity

Knight Lore Infinity is an experimental browser project 
that runs the ZX Spectrum game through JSSpeccy 3 while 
virtualizing the game map to have a new/larger map. 

The idea is to build a JavaScript-side sliding-window map 
virtualization system for Knight Lore, using only the 
original emulator view and runtime memory modification. 
The original game source code must not be modified.

This code is experimental. Use it at your own risk.

By Gustavo Patow
